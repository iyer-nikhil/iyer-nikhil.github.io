# My Website

![Screenshot](ss_sample.png)

This is a simple and minimalist website generated with Jekyll and designed to show off my portfolio.

The Theme features:

- Gulp
- SASS
- Sweet Scroll
- Particle.js
- BrowserSync
- Font Awesome and Devicon icons
- Google Analytics
